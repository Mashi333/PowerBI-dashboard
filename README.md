# 📊 Power BI Dashboard

A simple and interactive **Power BI Dashboard** built to visualize and analyze data through dynamic charts, KPIs, and filters. This project demonstrates how business intelligence tools can transform raw data into meaningful insights for better decision-making.

---

## 📌 Overview

This dashboard provides an easy-to-understand overview of the dataset using various visualizations. It enables users to monitor key metrics, identify trends, and explore data interactively.

---

## ✨ Features

- 📈 Interactive data visualizations
- 📊 KPI cards for important metrics
- 📉 Trend analysis with charts
- 🎯 Category-wise analysis
- 🔍 Slicers and filters for dynamic exploration
- 📋 Clean and user-friendly dashboard layout

---

## 🛠️ Built With

- **Microsoft Power BI Desktop**
- Power Query
- DAX (Data Analysis Expressions)

## 🚀 Getting Started

### Prerequisites

- Microsoft Power BI Desktop

### Open the Dashboard

1. Clone this repository.

```bash
git clone https://github.com/Mashi333/PowerBI-dashboard.git
```

2. Open the `.pbix` file using **Power BI Desktop**.

3. Refresh the data if required.
